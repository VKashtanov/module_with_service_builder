ROAD MAP , HOW TO CREATE A SIMPLE ENTITY VIA SERVICE BUILDER

1.Open INTELIJ IDEA
2.Choose service-builder as a type of the project
3.find service.xml in modules/crate-service/crate-service  where instead of "crate" may be any entity's name 
4.Compose your entity , add fields and other thing you need
5.RUN Service building generation by the command "./gradlew buildService" or via UI
6.Build your modules
7.deploy your modules
8.Add them to the folder "deploy"
9.Check wheter it started
