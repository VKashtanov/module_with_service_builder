# ROAD MAP , HOW TO CREATE A SIMPLE ENTITY VIA SERVICE BUILDER

1. Open INTELIJ IDEA
2. Choose service-builder as a type of the project
3. Find service.xml in modules/crate-service/crate-service  where instead of "crate" may be any entity's name 
4. !NOTE: #### Before creating you own service.xml , get aqcuainted with it in the project here.
5. Compose your entity , add fields and other thing you need
6. RUN Service building generation by the command "./gradlew buildService" or via UI
7. Build your modules
8. Deploy your modules
9. Add them to the folder "deploy"
10. Check wheter it started
